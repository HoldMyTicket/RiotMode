What is RiotMode?

RiotMode is a simple group of components built for Riot.js. They are currently a work in progress, but feel free to submit pull requests :)

Why?

Why not? We like Riot and thought it would be nice for a simple component library for others too.

How does it work?

Just make sure to import riot.js and riot.mode.js into your project. Then you can mount tags. 

<html>
<head>
  
</head>
<body>

  <rg-autocomplete></rg-autocomplete>

  <script src="riot.js"></script>
  <script src="riot.mode.js"></script>
  <script type="text/javascript">
    riot.mount('rg-autocomplete');
  </script>

</body>
</html>
